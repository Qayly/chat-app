# Chat Application

## Overview
This chat application is designed to facilitate one-on-one messaging between users. Built using a combination of Android, React, Java, and Firebase, the app leverages WebSocket technology to enable real-time communication.

## Features
- **Real-Time Messaging:** Users can send and receive messages instantly through WebSocket support.
- **Cross-Platform:** Built with React for the web and Android for mobile.
- **Firebase Integration:** Utilizes Firebase for backend services, including authentication and database management.

## Tech Stack
- **Frontend:** React (for web)
- **Mobile:** Android (Java)
- **Backend:** Firebase
- **Real-Time Communication:** WebSockets

## Getting Started
1. Clone the repository.
2. Set up Firebase and configure your project.
3. Run the appropriate build for your platform (Android or Web).
